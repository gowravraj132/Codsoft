# Task 1 - Personal Portfolio Website

This project is a personal portfolio website created using HTML and CSS. It helps showcase personal information, technical skills, and project work in a clean and professional manner.
Creating a personal portfolio using HTML and CSS is a popular beginner web development project. This website provides a structured layout to introduce yourself, highlight your skills, display projects, and share contact details. It is ideal for learning basic web page structure, styling, and layout design.

# Task 2 - Landing page

This project is a landing page website built using HTML and CSS. It focuses on creating a visually appealing and well-structured web page while strengthening fundamental front-end skills.
A landing page is a great way to practice core web development concepts. Through this project,learned how to structure content using sections and columns, organize layouts, and design headers and footers. The project encourages creativity while maintaining a clean and user-friendly design.

# Task 3 - Calculator

This project is a basic calculator application built using HTML, CSS, and JavaScript. It provides an interactive user interface that allows users to perform simple arithmetic operations such as addition, subtraction, multiplication, and division. It includes a display screen to show user input and calculation results, along with buttons for numeric values and arithmetic operations.
